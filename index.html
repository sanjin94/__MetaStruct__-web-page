Welcome to MetaStruct

MetaStruct is an innovative project focused on automating the translation of Industry Foundation Classes (IFC) to Finite Element Method (FEM) models in construction engineering. Our goal is to enhance accuracy and efficiency in construction simulations.

Project Roadmap

Months 1-2: Initial Research and Analysis

Conduct thorough research on existing IFC to FEM translation methods and tools.

Months 3-5: IFC to FEM Python Tool Development

Create Python tools for IFC file preparation, focusing on error detection and correction.

Begin initial testing on sample IFC files to assess tool effectiveness.
